# Discovery 调研确认模板

## 一、调研问题

```text
需要确认什么。
```

## 二、调研范围

```text
源码路径：
官方文档：
外部项目：
其他来源：
```

## 三、Confirmed

```text
已经确认的事实。
```

## 四、Assumption

```text
当前采用的合理假设。
```

## 五、Unknown

```text
仍未确认的问题。
```

## 六、证据来源

```text
文件路径、链接、命令输出或其他证据。
```

## 七、对设计的影响

```text
这些确认结果如何影响后续设计。
```
