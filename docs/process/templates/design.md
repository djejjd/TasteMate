# 设计文档模板

## 一、当前结论

```text
用一段话说明当前推荐方案。
```

## 二、背景与问题

```text
问题是什么，为什么现在要解决。
```

## 三、目标

```text
当前阶段要做到什么。
```

## 四、非目标

```text
当前阶段明确不做什么。
```

## 五、数据流

```text
输入 -> 处理 -> 输出
```

## 六、模块边界

```text
模块 A：
模块 B：
模块 C：
```

## 七、接口设计

```json
{
  "input": {},
  "output": {}
}
```

## 八、错误与降级

```text
错误场景：
降级方式：
```

## 九、成本与性能

```text
模型调用：
存储：
运行成本：
```

## 十、风险与应对

```text
风险：
应对：
```

## 十一、验收标准

```text
A-001
描述：
验证方式：
通过条件：
失败条件：
```

## 十二、后续迭代

```text
后续能力：
触发条件：
```
