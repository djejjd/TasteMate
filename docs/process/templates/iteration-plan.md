# 迭代规划模板

## 一、迭代名称

```text
迭代 N：名称
```

## 二、目标

```text
当前迭代要验证或交付什么。
```

## 三、范围

```text
本迭代包含什么。
```

## 四、不做事项

```text
本迭代明确不做什么。
```

## 五、验收标准

```text
A-001
描述：
验证方式：
通过条件：
失败条件：
```

## 六、主要风险

```text
风险：
影响：
```

## 七、风险应对

```text
应对：
降级：
```

## 八、分支或 Worktree 策略

```text
是否需要独立分支或 worktree：
原因：
```

## 九、进入下一迭代条件

```text
条件：
```
