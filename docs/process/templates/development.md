# 开发文档模板

## 适用阶段

```text
Development Spec，在 Design 通过后、Plan 之前编写或更新。
```

## 一、开发原则

```text
当前实现必须遵循的原则。
```

## 二、技术形态

```text
运行方式、依赖方式、集成方式。
```

## 三、目录结构

```text
path/
```

## 四、核心模块

```text
模块：
职责：
依赖：
```

## 七、数据结构


```text

```
